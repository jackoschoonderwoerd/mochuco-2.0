Welcome to Mochuco

You are currently in the administrator environment of the Mochuco app.

From here you can add and edit venues, items within the venues and adjust the properties of each of these items.

A venue has three properties:

  • A name (This name will not be shown to the user and is for reference only)
  • A logo (appears in the toolbar at the top of the window as soon as a user has scanned the qr-code that is assigned to one of the items in the venue)
  • A collection of items (each accesible by scanning a, for that specific item, unique qr-code)

Every item has two different properties:

  • An image
  • A collection of language specific properties

There are three language specific properties

  • A name
  • A description
  • An audio section

While the image for a specific item stays the same, the other three properties can be added or modified in accordance with a previously selected language

Only one item within each venue can be designated as 'main item'.

This main item will have a green background in the item-overview list

The only way for a user to reach this 'main item' is by clicking on the venue logo in the toolbar at the top of the window.

It's purpose is to give the user an idea about what the venue in general entails.

Any other item is meant to give specific information about a particular detail of the venue and can only be reached by scanning it's unique qr-code.

As soon as a user scans the qr-code assinged to a specific item. he will automatically be foreward to a page with the relevant information in the language of his choice.
